# DicePredict
語言：C#  
程式最後修改時間：2019/01/03  
透過以往的開骰數據  
每三個數字為一組  
前兩個數字表示上上一次及上一次開出的數字  
統計出前兩個數字出現後第三個數字出現什麼的機率最高  
  
驗證系統  
利用爬蟲爬取特定的網站再分析網站內是否有開放使用的訊息，達到類似序號驗證的效果  
爬取的網址為免費html伺服器：https://whiteeyeyan.neocities.org/
